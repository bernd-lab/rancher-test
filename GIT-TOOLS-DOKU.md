### git-delta - Syntax-Highlighted Diffs

**Was macht es?** Macht `git diff` sch√∂n lesbar mit Syntax-Highlighting, Side-by-Side-View und besserer Formatierung.

**Setup:**

**Konfiguration (empfohlen):**

**Beispiele:**

**Vorteile:**
- ‚úÖ Syntax-Highlighting f√ºr YAML, JSON, Go, etc.

**`.pre-commit-config.yaml` Beispiel f√ºr GitOps:**

```yaml
      - id: kubeval

  - repo: https://github.com/Yelp/detect-secrets

    hooks:
```

**Verwendung:**
